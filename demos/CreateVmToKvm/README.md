# 動作環境

* Node.js v1.12.x(v1.10.xでも動くと思います)
* KVMサーバ2台にsshできること
* その2台をubu1804-kvm001, ubu1804-kvm002で名前解決できること


# 使い方

## Webサーバ起動

```
$ node endpoint
```

ブラウザで `http://localhost:30117/kvms.html` にアクセス(少し時間がかかります)

# 仮想マシン構築コマンドを実行する

2台のKVMのうち、リソースに空きがある方(両方空いている場合はランダムにどちらかが選ばれる)にCentOS7の仮想マシンを作成する

```
$ node index.js
```
